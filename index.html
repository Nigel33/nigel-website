Guess a Word

Try to guess the hidden word! You only have as many tries as the amount of apples on the tree! Every incorrect guess removes an apple. Built with Javascript and jQuery.

Dynamic Animation

Add a shape you want from the selection provided and input in the starting and ending X & Y coordinates. Press "Start" and watch the shapes come to live! Built with Javascript and jQuery.

Inventory Order Form

Grocery List

Search Filtering

A search form which dynamically filters out selection based on the checkboxes selected. Items are indexed in a JSON file to dynamically pull in selected items. Built with JavaScript and jQuery.

Company Site

A full company site with 3 pages linked together. Renders a modal with an overlay when a team member's photo is clicked. Built with pure HTML and CSS.

Photo Gallery

A photo gallery that fades in a photo to the main frame depending on the selected photo. Uses CSS to hide elements and jQuery to reveal each photo selected.

Comic

Risk Tracker

Risk Tracker is a CMS for a hypothetical insurance company. Its an imitation of the project I was working with in the past with MS Access as a means of providing a more efficient, scalable solution. It is built with Rails adhering to the Agile development principles and is backed with a complete unit and integration test suite. It uses jQuery for some front end behavior.

This video gives a quick demonstration of the features of this application. Subsequent videos in this video series will provide a more detailed walkthrough of various implementation processess I employed while developing this application.
